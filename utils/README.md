# Utility Scripts

This folder contains various utility scripts for dataset management and testing.

## Files

- **display_image.py**: Displays image information and pixel values
- **countafimages.py**: Counts the number of images in each disease class after splitting
- **countogimages.py**: Counts the number of images in the cleaned dataset (same as x.py)
- **prepcheck.py**: Checks and validates preprocessed .npy files
- **tsting.py**: Tests loading and inspecting .npy file structure
- **delete_prepro.py**: Removes preprocessed data directory (use with caution!)
- **x.py**: Same as countogimages.py - counts images in cleaned dataset

## Purpose

These scripts help with:
- Dataset inspection and validation
- Image counting and statistics
- File format verification
- Data cleanup operations
